# 🎸 Freddie Mercury — Tribute Page

A newspaper-style tribute page dedicated to the legendary musician *Freddie Mercury*, built using HTML and CSS.

## 🔗 Live Demo
[https://maitrishivhare.github.io/OIBSIP-level2-task2/](https://maitrishivhare.github.io/OIBSIP-level2-task2/)

---

## 📌 About the Project

This is a tribute page created as part of the *OIBSIP (Oasis Infobyte Internship Program)* — Level 2, Task 2.

The page is styled in a classic *newspaper / editorial aesthetic*, featuring:
- A bold masthead header
- Hero image of Freddie Mercury
- Multi-column editorial layout
- Pull quotes and drop caps
- Photo gallery section
- Quick facts & greatest songs sidebar
- Fully responsive design

---

## 🛠️ Built With

- *HTML5* — Page structure and content
- *CSS3* — Styling, layout (CSS Grid), and responsiveness
- *Google Fonts* — Playfair Display, IM Fell English, Libre Baskerville
